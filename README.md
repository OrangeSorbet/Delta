# AWS Deployment README — Metals & Gems Price Predictor

## Project Overview
Predicting next-day prices (USD + INR) for precious metals and gemstones — Gold, Silver, Platinum, Copper, Diamond, Lab Diamond, Emerald, Ruby, Pearl — using a trained ML model. A 30-day rolling forecast is generated by feeding predictions back as input iteratively.

## Step 1 — EC2 Instance Launch

**Settings used:**
| Field | Value |
|---|---|
| AMI | Ubuntu 26.04 LTS, amd64 |
| Instance type | t3.micro (free tier eligible) |
| Key pair type | RSA |
| Key pair file | `metals-key.pem` |
| Storage | 20 GiB gp3 |

**Security group inbound rules:**
| Type | Port | Source |
|---|---|---|
| SSH | 22 | 0.0.0.0/0 |
| HTTP | 80 | 0.0.0.0/0 |
| HTTPS | 443 | 0.0.0.0/0 |
| Custom TCP | 8000 | 0.0.0.0/0 |
